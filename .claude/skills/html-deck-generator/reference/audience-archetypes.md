# Audience Archetype Playbook

Use these templates to quickly tailor HTML decks once baseline intent is captured. Each archetype lists:
- **Audience signals** to confirm you’re in the right profile
- **Slide stack** that maps to the core narrative beats
- **Copy tone** guidelines to maintain cohesion
- **Theming nudges** (CSS variable suggestions) to align visuals
- **Probing prompts** to resolve common gaps in the brief

## Investor Pitch
- **Signals:** Fundraising goals, investor meetings, traction metrics requests, references to runway or round size.
- **Slide Stack:** Hero → traction metrics → market size → solution/product → roadmap → financial ask → CTA.
- **Copy Tone:** Confident, concise, numbers-first, highlight momentum.
- **Theming Nudges:** `--primary-gradient: linear-gradient(135deg, #ff6b35, #ff8c42);` `--surface-color: #0b0b0f;` `--card-surface: rgba(255,255,255,0.08);`
- **Probing Prompts:** “What fundraising round and amount should we anchor on?” “Any standout KPIs or traction metrics to spotlight?” “Do we call out specific investors or partners?”

## Internal Onboarding
- **Signals:** New hire enablement, culture overview, process docs, references to onboarding portal or playbooks.
- **Slide Stack:** Hero → mission & values → organization snapshot → process timeline → resources directory → CTA (e.g., onboarding checklist).
- **Copy Tone:** Warm, inclusive, supportive.
- **Theming Nudges:** `--primary-gradient: linear-gradient(135deg, #4facfe, #38f9d7);` `--surface-color: #101522;` `--card-surface: rgba(255,255,255,0.12);`
- **Probing Prompts:** “What’s the day-one mission statement you want to reinforce?” “Which resources or teams must be linked?” “Do we need compliance or HR reminders?”

## Enterprise Sales
- **Signals:** Stakeholder consensus, pilot results, ROI proof, procurement steps.
- **Slide Stack:** Hero → problem framing → solution architecture → proof points/testimonials → ROI model → implementation timeline → CTA.
- **Copy Tone:** Consultative, value-oriented, precise.
- **Theming Nudges:** `--primary-gradient: linear-gradient(135deg, #6a6fff, #4b4efc);` `--surface-color: #080b13;` `--card-surface: rgba(255,255,255,0.1);`
- **Probing Prompts:** “Which stakeholders must we convince and what do they care about most?” “Any security/compliance assurances required?” “Is there a target go-live date to emphasise?”

## Conference Keynote
- **Signals:** Speaker references, visionary themes, trend analysis, inspirational anecdotes.
- **Slide Stack:** Hero → narrative quote → megatrend data → product/vision showcase → roadmap → community/CTA.
- **Copy Tone:** Inspirational, thought-leadership, story-driven.
- **Theming Nudges:** `--primary-gradient: linear-gradient(135deg, #ff7eb3, #ff758c);` `--surface-color: #07070b;` `--card-surface: rgba(255,255,255,0.08);`
- **Probing Prompts:** “What’s the headline quote or mantra for this talk?” “Which trend or data point anchors the story?” “Any live demo moments or multimedia cues?”

## Product Update
- **Signals:** Release notes, feature launch, adoption nudges, changelog mentions.
- **Slide Stack:** Hero → feature rundown → UX screenshots/demos → adoption metrics → customer feedback → CTA (upgrade, sign in, enable feature).
- **Copy Tone:** Energetic, clear, benefit-driven.
- **Theming Nudges:** `--primary-gradient: linear-gradient(135deg, #ffb347, #ffcc33);` `--surface-color: #0f111a;` `--card-surface: rgba(255,255,255,0.14);`
- **Probing Prompts:** “What’s the headline feature or improvement?” “Any launch timeline or phased rollout details?” “Should we highlight upgrade instructions or pricing changes?”

## Blending Archetypes
When requests touch multiple archetypes, pick one as the dominant pattern and borrow slide beats from the secondary. Document the mix explicitly in your intent summary so downstream collaborators understand the rationale.
