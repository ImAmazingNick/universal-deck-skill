# Brand Color Reference

Complete color palette definitions for brand consistency.

## Primary Colors

### Corporate Blue
- **Hex**: `#0066CC`
- **RGB**: `rgb(0, 102, 204)`
- **Usage**: Primary brand color, headers, CTAs
- **Accessibility**: WCAG AA compliant on white

### Startup Green
- **Hex**: `#00D4AA`
- **RGB**: `rgb(0, 212, 170)`
- **Usage**: Secondary actions, highlights
- **Accessibility**: WCAG AA compliant on dark backgrounds

### Tech Purple
- **Hex**: `#8B5CF6`
- **RGB**: `rgb(139, 92, 246)`
- **Usage**: Accents, special emphasis
- **Accessibility**: WCAG AA compliant on white

## Neutral Colors

### Background
- **White**: `#FFFFFF` - Primary background
- **Light Gray**: `#F9FAFB` - Secondary background

### Text
- **Dark Gray**: `#1A1A1A` - Primary text
- **Medium Gray**: `#6B7280` - Secondary text
- **Light Gray**: `#9CA3AF` - Muted text

### Borders
- **Light Gray**: `#E5E7EB` - Default borders
- **Medium Gray**: `#D1D5DB` - Strong borders

## Usage Guidelines

### Contrast Ratios
- Text on white: Minimum 4.5:1
- Large text: Minimum 3:1
- UI components: Minimum 3:1

### Color Combinations
- Primary + White: ✅ Recommended
- Secondary + Dark: ✅ Recommended
- Accent + White: ✅ Recommended
- Avoid: Primary + Secondary (low contrast)

## Theme Mapping

Brand colors map to marketing-deck-generator themes:
- Corporate Blue → `corporate-blue` theme
- Startup Green → `startup-green` theme
- Tech Purple → `tech-purple` theme
