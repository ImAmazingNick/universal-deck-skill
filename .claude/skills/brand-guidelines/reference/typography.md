# Brand Typography Reference

Typography standards for consistent brand communication.

## Font Families

### Headings
- **Primary**: `Space Grotesk`
- **Fallback**: `Inter`, `-apple-system`, `BlinkMacSystemFont`, `sans-serif`

### Body Text
- **Primary**: `Inter`
- **Fallback**: `-apple-system`, `BlinkMacSystemFont`, `sans-serif`

### Monospace
- **Primary**: `JetBrains Mono`
- **Fallback**: `'Courier New'`, `monospace`

## Font Sizes

### Headings
- **5xl**: 48px / 3rem - Hero titles
- **4xl**: 36px / 2.25rem - Section titles
- **3xl**: 30px / 1.875rem - Subsection titles
- **2xl**: 24px / 1.5rem - Card titles
- **xl**: 20px / 1.25rem - Small headings
- **lg**: 18px / 1.125rem - Large body

### Body Text
- **base**: 16px / 1rem - Default body
- **sm**: 14px / 0.875rem - Small text
- **xs**: 12px / 0.75rem - Captions

## Font Weights

- **Light**: 300
- **Regular**: 400
- **Medium**: 500
- **Semibold**: 600
- **Bold**: 700

## Line Heights

- **Tight**: 1.25 - Headings
- **Normal**: 1.5 - Body text
- **Relaxed**: 1.75 - Long-form content

## Usage Guidelines

### Headings
- Use Space Grotesk for all headings
- Maintain clear hierarchy (h1 > h2 > h3)
- Use appropriate weights (Bold for h1, Semibold for h2)

### Body Text
- Use Inter for all body text
- Maintain readable line height (1.5)
- Use Regular weight for body, Medium for emphasis

### Code
- Use JetBrains Mono for code blocks
- Maintain monospace formatting
- Use appropriate sizing (sm or base)

## Integration

Typography settings align with marketing-deck-generator themes and are automatically applied when generating presentations.
