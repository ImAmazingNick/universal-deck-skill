# Brand Guidelines

Maintain consistent branding across all generated content, presentations, and marketing materials.

## When to Use

Use this skill when you need to:
- Ensure consistent branding across presentations and documents
- Apply brand colors, typography, and visual elements
- Validate content against brand guidelines
- Generate branded templates and materials
- Maintain visual consistency across multiple deliverables

## Brand Standards

### Color Palette

**Primary Colors:**
- Primary: `#0066CC` (Corporate Blue)
- Secondary: `#00D4AA` (Startup Green)
- Accent: `#8B5CF6` (Tech Purple)

**Neutral Colors:**
- Background: `#FFFFFF` (White)
- Foreground: `#1A1A1A` (Dark Gray)
- Muted: `#6B7280` (Medium Gray)
- Border: `#E5E7EB` (Light Gray)

### Typography

**Headings:**
- Font Family: `Space Grotesk`, `Inter`, system fonts
- Sizes: 2xl (24px), xl (20px), lg (18px)

**Body Text:**
- Font Family: `Inter`, system fonts
- Size: Base (16px), lg (18px)

**Monospace:**
- Font Family: `JetBrains Mono`, monospace
- Used for code and technical content

### Visual Elements

**Logo Usage:**
- Minimum clear space: 2x logo height
- Do not distort or rotate logos
- Maintain aspect ratio

**Spacing:**
- Base unit: 8px
- Consistent margins and padding
- Grid system: 12 columns

**Images:**
- High resolution (minimum 300 DPI for print)
- Consistent aspect ratios
- Proper alt text for accessibility

## Application Guidelines

### For Presentations

When generating presentations:
1. Use brand colors from theme system
2. Apply consistent typography
3. Maintain logo placement standards
4. Follow spacing guidelines

### For Documents

When creating documents:
1. Use brand fonts and sizes
2. Apply color palette consistently
3. Include proper branding elements
4. Maintain visual hierarchy
